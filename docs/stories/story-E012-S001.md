# 类型化看板切换

**ID:** STORY-E012-S001  
**史诗:** E012 类型化看板与统计  
**优先级:** P2  
**故事点:** 3

## 用户故事

作为管理层，我希望能按资产类型切换看板视图，以便分别查看固定资产、不动产和低值易耗品的运营情况。

## 验收标准

- [ ] 看板可按资产类型切换。
- [ ] 不同类型显示不同的核心指标卡片。
- [ ] 数据口径与台账一致。
- [ ] 固定资产、不动产、低值易耗品至少都能看到自己的基础看板。
- [ ] 看板页面不需要为每种类型单独复制一套壳。

## 技术说明

- 看板应复用同一套页面壳，类型差异通过配置和指标定义区分。
- 该能力适合作为系统运营入口，不建议过早细化成太多图表。
- 数据接口需保留类型过滤能力。

## 前置依赖

- `E009` 资产类型底座与差异化框架
- 现有 `dashboard` 能力

- [ ] 文档同步更新
- [ ] 代码审查通过
