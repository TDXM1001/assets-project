# 类型化待办与异常汇总

**ID:** STORY-E012-S003  
**史诗:** E012 类型化看板与统计  
**优先级:** P2  
**故事点:** 3

## 用户故事

作为管理者，我希望统一看到不同类型资产的待办和异常，以便集中处理审批、维修、盘点、续租和补领等事项。

## 验收标准

- [ ] 支持按资产类型汇总待办。
- [ ] 支持按异常类型筛选。
- [ ] 能跳转到对应业务页处理事项。
- [ ] 待办与异常可以在同一入口统一查看。
- [ ] 不同类型的待办文案和状态定义清晰。

## 技术说明

- 该故事适合作为系统运营闭环的统一入口。
- 待办和异常可以优先做列表化汇总，后续再扩展更复杂的工作台。
- 不同类型的异常定义应来自类型规则，而不是硬编码在看板里。
