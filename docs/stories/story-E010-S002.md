# 不动产权属与证照管理

**ID:** STORY-E010-S002  
**史诗:** E010 不动产管理  
**优先级:** P1  
**故事点:** 3

## 用户故事

作为资产管理员，我希望能够维护不动产权属和证照信息，以便跟踪产权归属、共有关系和证照到期风险。

## 验收标准

- [ ] 支持录入产权证号、权利人、共有权人和证照有效期。
- [ ] 支持证照附件与扫描件关联。
- [ ] 证照到期或即将到期时能够被识别为风险。
- [ ] 权属信息可在详情中被稳定查看。
- [ ] 权属信息与主档分区清晰，不混入固定资产字段。

## 技术说明

- 权属和证照更适合独立分区，不建议与一般使用信息混在同一块表单里。
- 到期预警可以先做基础规则，后续再接更复杂的提醒和统计。
- 证照附件建议复用通用附件能力。

- 现有附件能力

- [ ] 证照字段保存和回显正常
- [ ] 文档同步更新
