# 资产事件流水

**ID：** STORY-E004-S001  
**史诗：** E004 运营视图  
**优先级：** Should Have  
**故事点：** 3

## 用户故事

作为审计或管理人员，我希望查询资产事件流水，以便追溯资产每一次关键变更的前后值、操作者和来源单据。

## 验收标准

- [x] 可以按资产、单据和事件类型查询流水。
- [x] 可以查看关键字段的前后值和来源单据。
- [x] 可以导出流水数据。
- [x] 访问流水需要满足业务权限和数据范围控制。

## 技术说明

- 沿用 `asset/event` 模块完成只读查询与导出。
- 流水页面要支持资产维度的时间线查看。
- 只读功能也要保持权限和数据范围一致。
- 当前实现采用“列表页做完整检索、抽屉展示最近 20 条时间线”的分层方式，满足审计查询和快速查看两类场景。
