# 类型字段模板与渲染规则

**ID:** STORY-E009-S002  
**史诗:** E009 资产类型底座与差异化框架  
**优先级:** P0  
**故事点:** 5

## 用户故事

作为资产管理员，我希望不同资产类型能展示不同的字段、校验规则和组件，以便录入时只看到当前类型需要的内容。

## 验收标准

- [ ] 切换资产类型后，页面能渲染不同的字段区块。
- [ ] 字段支持必填、只读、默认值、排序、分组和组件类型配置。
- [ ] 模板支持版本信息，且版本变化不会破坏已存在资产数据。
- [ ] 模板切换后，已填写的字段能够按规则保留或清空。
- [ ] 前端渲染规则与后端模板定义保持一致。

## 技术说明

- 该故事建议沿用受控模板，而不是完全自由的低代码表单引擎。
- 字段模板应优先复用既有分类模板能力，避免重复设计模板数据结构。
- 核心字段继续保留在主档里，类型差异字段通过模板接入。
- 前端渲染层只负责渲染和校验，不承载复杂业务规则判断。
