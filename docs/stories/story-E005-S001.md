## 用户故事

作为资产管理员，我希望可以给单据、资产或盘点记录上传和查看附件，以便保留图片、说明和证明材料。

- [x] 可以按业务类型和业务 ID 上传附件。
- [x] 可以查询和删除附件。
- [x] 附件列表可以在前端抽屉或弹窗中查看。
- [x] 附件能力受到业务权限控制。

## 技术说明

- 沿用 `asset/attachment` 模块完成附件上传、列表和删除。
- 先保证主流程可用，再补充预览和信息密度优化。
- 上传文件的业务归属必须可追踪。
- 当前实现复用同一个附件抽屉组件，在单据、资产和维修页统一查看与维护附件。
