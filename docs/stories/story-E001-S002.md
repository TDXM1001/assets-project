- [x] 可以新增、编辑、停用存放位置。
- [x] 位置树层级正确展示，支持部门、区域、地点等层次。

- 沿用 `asset/location` 模块骨架完成位置管理。
- 位置树要兼容前台选择器和后端查询口径。
- 删除或停用时要处理引用关系，避免脏数据。
