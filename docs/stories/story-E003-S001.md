# 盘点任务与差异处理

**ID：** STORY-E003-S001  
**史诗：** E003 盘点与差异  
**优先级：** Must Have  
**故事点：** 5

## 用户故事

作为资产管理员，我希望创建和处理盘点任务，以便系统能够识别账实差异并回写资产主档。

## 验收标准

- [x] 可以创建、编辑、开始、结束和查询盘点任务。
- [x] 可以批量生成盘点明细并在盘点过程中处理差异。
- [x] 盘点结果可以回写资产位置、责任人和状态。
- [x] 差异处理结果会写入事件流水，便于追溯。

## 技术说明

- 沿用 `asset/inventory` 模块完成盘点任务骨架。
- 开始、扫描、结束和差异处理必须保持事务一致性。
- 需要支持按部门、位置、分类等维度生成盘点任务。
- 当前实现采用“先结束盘点，再处理差异”的状态流转方式，与后端状态机保持一致。
