# 不动产处置与退出

**ID:** STORY-E010-S006  
**史诗:** E010 不动产管理  
**优先级:** P2  
**故事点:** 5

## 用户故事

作为资产管理员，我希望能对不动产执行出售、转让、注销、拆除或退出使用，以便完整关闭不动产生命周期。

## 验收标准

- [ ] 支持创建不动产处置或退出单据。
- [ ] 支持记录处置原因、方式、结果和附件。
- [ ] 处置后状态变为退出或已处置，不能再按在用资产流转。
- [ ] 处置结果可追溯到事件流水。
- [ ] 处置语义与固定资产报废语义保持区分。

## 技术说明

- 不动产处置通常比一般资产报废更重，建议保留更明确的流程语义。
- 该故事应为后续财务、权属或合同扩展留接口，但第一版不做重 BPM。
- 处置完成后应同步影响看板和统计口径。
