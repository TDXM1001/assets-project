# 资产台账管理

**ID：** STORY-E001-S003

## 用户故事

作为资产管理员，我希望维护资产台账，以便能够按编码、分类、责任人和位置查询资产，并支持基础导出能力。

## 验收标准

- [x] 可以新增、编辑、删除和查询资产台账。
- [x] 资产编码唯一，重复编码会被拦截并提示。
- [x] 可以按编码、分类、责任人、位置筛选台账。
- [x] 可以导出资产台账列表。

## 技术说明

- 列表页要兼容权限范围和空态、异常态。
