# 低值易耗品入库与批次管理

**ID:** STORY-E011-S002  
**史诗:** E011 低值易耗品管理  
**优先级:** P1  
**故事点:** 5

## 用户故事

作为资产管理员，我希望能够批量入库低值易耗品并记录批次，以便管理供应来源、有效期和库存变化。

## 验收标准

- [ ] 支持批量入库。
- [ ] 支持记录批次、数量、供应信息和有效期。
- [ ] 入库后库存可以正确回写。
- [ ] 入库记录可以追溯到来源单据。
- [ ] 批次信息可在列表和详情中查看。

## 技术说明

- 低值易耗品的关键不在“资产去向”，而在“库存来源和数量变化”。
- 批次字段建议尽早保留，便于后续库存和临期管理。
- 入库流程可以先简单，后续再考虑导入或更复杂的供应链对接。
