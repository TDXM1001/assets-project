# 类型化统计指标定义

**ID:** STORY-E012-S002  
**史诗:** E012 类型化看板与统计  
**优先级:** P2  
**故事点:** 3

## 用户故事

作为管理者，我希望不同资产类型有自己的统计指标定义，以便看到真正有意义的运营数据。

## 验收标准

- [ ] 固定资产、不动产、低值易耗品分别有独立的指标定义。
- [ ] 指标来源清晰，且与台账或业务流水一致。
- [ ] 切换资产类型后，统计卡片能正确变化。
- [ ] 指标定义可维护，不需要每次都改页面代码。
- [ ] 统计口径不会混用不同类型的字段。

## 技术说明

- 固定资产重点看在用、维修、待盘点和待审批。
- 不动产重点看空置、出租、巡检、维修和证照风险。
- 低值易耗品重点看库存、消耗、周转和预警。
