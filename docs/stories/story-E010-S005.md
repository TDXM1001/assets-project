# 不动产维修保养与巡检

**ID:** STORY-E010-S005  
**史诗:** E010 不动产管理  
**优先级:** P2  
**故事点:** 8

## 用户故事

作为资产管理员，我希望能对不动产执行维修、保养、巡检和整改，以便处理房产和空间的维护问题，并保持资产可用性和安全性。

## 验收标准

- [ ] 支持维修保养单或巡检单创建。
- [ ] 支持记录问题描述、费用、附件、处理结果和完工状态。
- [ ] 支持关联到具体空间、楼栋、房间或设施。
- [ ] 巡检和维修结果能回写到事件流水。
- [ ] 该模块语义区别于固定资产维修，名称和字段更贴近不动产场景。

## 技术说明

- 不动产的维护建议分成保养、维修、巡检和整改四类语义。
- 如果后续要接改造工程，当前数据结构应保留工程性质的扩展位。
- 该能力可以复用现有维修模块的壳，但业务语义应独立配置。
