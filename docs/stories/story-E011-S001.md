# 低值易耗品主档与分类

**ID:** STORY-E011-S001  
**史诗:** E011 低值易耗品管理  
**优先级:** P1  
**故事点:** 5

## 用户故事

作为资产管理员，我希望能够登记低值易耗品的主档和分类，以便管理规格、单位、数量和库存。

## 验收标准

- [ ] 支持新增、编辑和查询低值易耗品主档。
- [ ] 主档至少包含名称、规格、单位、数量、批次和库存字段。
- [ ] 支持分类和状态管理。
- [ ] 支持临期或低库存信息的基础展示。
- [ ] 与固定资产、不动产主档可以并存，不互相干扰。

## 技术说明

- 低值易耗品应当更偏数量型台账，不建议沿用固定资产那套强设备语义字段。
- 该模块的字段应尽量轻量，核心是数量、单位、批次和库存。
- 主档应为后续批量入库和领用流程提供入口。
