# 导入导出与权限收口

**ID：** STORY-E005-S002  
**史诗：** E005 增强与收口  
**优先级：** Should Have  
**故事点：** 5

## 用户故事

作为资产管理人员，我希望完善导入导出和权限收口能力，以便把系统从“能用”收口到“比较稳”。

## 验收标准

- [x] 主要列表页支持导出，导出字段和查询口径一致。
- [x] 资产台账支持导入模板和导入流程入口。
- [x] 菜单权限、按钮权限和数据范围命名统一。
- [x] 不同角色进入系统后，页面、按钮和数据范围符合设计。

## 技术说明

- 这一项主要收口权限、导入导出和空态/异常态表现。
- 需要统一 `asset_admin`、`asset_dept_manager`、`asset_user`、`asset_auditor` 的前后端口径。
- 导入和导出要给出明确的校验提示。
- 资产管理员保留导入入口，部门主管和审计角色仅保留查询/导出等必要权限。

- [x] 完成空态、错误态和加载态补齐。
