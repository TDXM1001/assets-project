1. [资产管理系统-产品方案.md](./资产管理系统-产品方案.md)
- 说明系统要解决什么问题、核心业务模块、角色设计和业务流程。
- 适合先读，用来统一产品和业务理解。

2. [资产管理系统-实施方案.md](./资产管理系统-实施方案.md)
- 说明表结构拆分、菜单设计、权限标识、接口清单和推荐开发顺序。

- 适合在真正写 MySQL DDL 之前阅读。

4. [资产管理系统-菜单与权限设计.md](./资产管理系统-菜单与权限设计.md)
- 说明菜单树、按钮权限、角色边界、数据范围和 `sys_menu` 初始化策略。
- 适合在写菜单初始化 SQL 和前后端权限联调前阅读。

5. [资产管理系统-接口分组与开发计划.md](./资产管理系统-接口分组与开发计划.md)
- 说明哪些接口适合先写标准 CRUD 骨架、哪些必须按流程手写，以及建议的开发阶段和交付顺序。
- 适合在真正开始实现前阅读。

6. [资产管理系统-业务流程演练稿.md](./资产管理系统-业务流程演练稿.md)

- 适合在写正式 DDL 和开始编码前阅读。

7. [资产管理系统-MySQL-DDL草案.md](./资产管理系统-MySQL-DDL草案.md)
- 把核心业务表推进到接近可执行的 MySQL DDL 形式，便于后续转成正式 SQL 文件。
- 适合在真正建表前阅读。

8. [资产管理系统-sys_menu初始化SQL草案.md](./资产管理系统-sys_menu初始化SQL草案.md)
- 把资产管理模块的目录、页面菜单和按钮权限推进到接近可执行的 `sys_menu` 初始化 SQL 形式。
- 适合在真正写菜单初始化脚本前阅读。

9. [资产管理系统-sys_role_menu授权草案.md](./资产管理系统-sys_role_menu授权草案.md)

- 适合在真正分配角色权限前阅读。

10. [资产管理系统-后端代码骨架清单.md](./资产管理系统-后端代码骨架清单.md)
- 在“不使用代码生成器”的前提下，梳理资产模块后端应该落到哪些目录、类和接口。
- 适合在真正开始写后端代码前阅读。

11. [资产管理系统-前端页面与路由骨架清单.md](./资产管理系统-前端页面与路由骨架清单.md)

- 适合在真正开始写前端页面前阅读。

12. [资产管理系统-开发任务拆分清单.md](./资产管理系统-开发任务拆分清单.md)

- 适合在真正开始排期和开工前阅读。
